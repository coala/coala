# landing-frontend
coala Landing Page - https://gitlab.com/coala/landing is needed as backend


## Usage

To run locally:

    $ git clone https://github.com/coala/landing-frontend.git
    $ cd landing-frontend
    $ python2 -m SimpleHTTPServer 8080


Open http://localhost:8080 in your browser.
